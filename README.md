# Superhero APP

Find and compare your favourite heroes from all universes.

## Technologies:

- React
- Hooks
- axios
- Sass
- React Router

## Setup

App isn't live yet.
To run this project, install it locally using npm:

```
$ cd ../your-folder
$ npm install
$ npm start
```

### All data obtained from [Superhero API](https://www.superheroapi.com/)
